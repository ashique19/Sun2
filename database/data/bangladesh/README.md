# Bangladesh administrative locations

Districts and police stations (upazilas / thanas) sourced from the
[thikkit](https://github.com/rockyhaque/thikkit-npm) open dataset, which compiles
data from the Bangladesh Bureau of Statistics and metropolitan police authorities.

- `divisions.json` — 8 divisions
- `districts.json` — 64 districts
- `units.json` — 603 upazilas and metropolitan thanas

Seed with:

```bash
php artisan locations:seed
```
